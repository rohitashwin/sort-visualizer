# Sort Visualizer

Shows how various sorting algorithms are sorted.

### Algorithms

- Bubble Sort
- Insertion Sort
- Merge Sort (inplace)
- Merge Sort (asynchronous, inplace)
- Quicksort
- Quicksort (asynchronous)
- Heap Sort
- Counting Sort
